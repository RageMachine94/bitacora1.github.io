HTML

Ideas Generales

    Estructura básica de un documento HTML
  • Un documento HTML tiene una estructura estándar que incluye varios elementos clave.
    Estiquetas HTML comunes
  • Las etiquetas se utilizan para definir diferentes tipos de contenido. Algunas de las más comunes son:
  • Encabezados: Se usan para definir títulos en diferentes niveles
  • Párrafos: Se utilizan para definir bloques de texto.
  • Enlaces: Permiten crear hipervínculos que llevan a otras páginas o sitios web
  • Imágenes: Se usan para insertar imágenes en la página
  • Listas: Para crear listas numeradas o con viñetas.

Visto en clase

Para empezar a escribir en HTML hay que tener en cuenta que la estructura se mide de diferente forma, para empezar a escribir primero tendrás que escribir etiquetas predefinidas, la estructura de la sintaxis correcta es html, head, body, La primera indica el comienzo de la página; la segunda es la cabecera y contiene información descriptiva del sitio; y la tercera es la que abarca todos los elementos visibles dentro de la página. Una vez entendido esto veremos como ejemplificarlo en el mismo programa de Visual Studio Code. Es preferible usar este software ya que nos simplifica el uso de etiquetas en él, como podremos ver una vez configurado par escribir HTML como lo vimos antes, solo tendrás que escribir html, seguido de ello se desplegara una pestaña en la cual nos mostrara diferentes opciones para autocompletar texto, nosotros elegiremos la segunda opción que es html:5. Una vez hecho esto, nos autocompletara el texto con sus respectivas etiquetas vistas anteriormente listas para su uso.

Resumen

HTML es el lenguaje de marcado utilizado para definir la estructura y el contenido de las páginas web. A través de etiquetas y atributos, puedes organizar texto, imágenes, enlaces, listas, formularios, y mucho más. Además, puedes combinarlo con CSS para el diseño visual y con JavaScript para la interactividad.
